# Game Launcher

Локальный лаунчер для Windows, написанный на Rust с использованием GUI-библиотеки iced.

Проект находится в ранней alpha-стадии. Интерфейс и логика могут меняться, часть функций может работать нестабильно.

## Возможности

- Добавление игр/приложений в лаунчер
- Отображение элементов в интерфейсе
- Запуск добавленных программ
- Базовая работа с Windows `.lnk`-ярлыками
- Локальное хранение данных
- Portable-запуск без установки
